# Konsol Tabanlı Metin Editörü (Mini Notepad++)

Bu proje, C++ programlama dili kullanılarak geliştirilen konsol tabanlı bir metin editörü uygulamasıdır. Kullanıcıların terminal ortamında metin dosyaları oluşturmasına, düzenlemesine ve kaydetmesine olanak sağlar.

## Özellikler

- Metin yazma ve düzenleme
- Dosya açma / kaydetme
- İmleç hareketleri
- Klavye kısayolları desteği
- Satır bazlı düzenleme
- Hızlı ve hafif konsol arayüzü

## Kullanılan Teknolojiler

- C++
- Visual Studio
- Dosya İşlemleri
- Konsol Programlama

## Proje Amacı

Temel bir metin editörünün çalışma mantığını anlamak, kullanıcı etkileşimli konsol uygulamaları geliştirmek ve C++ dilinde uygulamalı proje deneyimi kazanmaktır.
